    • Tarjeta gráfica GeForce 8800 o equivalente (GPU dedicada con 512 MB de memoria gráfica (VRAM) o más)

    • Windows XP con el último Service Pack instalado, Windows Vista con el último Service Pack instalado o Windows 7
